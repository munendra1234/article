Approach :

We have to print a pattern which resembles the shape of ‘W’.

In this design, we observe that we need stars in all the rows of the first and last column; and on the diagonal positions, provided diagonal positions are below the middle row (inclusive).

So, we know the condition for the first and last row will be

‘j’ == 1 || ‘j’ == ‘n’ and condition for diagonal values are ‘i’ ==’j’ || ‘i’ +’j’ == n + 1

But in diagonal, we place another check, that is , the middle row is crossed ( ‘i’ > n/2). Any place else, then we need to print whitespaces.

import java.util.*; public class Main{ public static void main(String[] args) { Scanner scn = new Scanner(System.in); int n = scn.nextInt(); for(int i = 1;i<=n;i++) { for(int j =1;j<=n;j++) { if( j == 1 || j == n) //if first or last column { System.out.print("*\t"); } else if(i > n/2 && (i ==j || i +j == n + 1)) // part of either diagonal below middle row { System.out.print("*\t"); } else { System.out.print("\t"); } } System.out.println(); } } }
java; true";
