In this article, we are going to solve The problem of saddle price. The saddle price is defined as the least value in the row and the maximum value in the column.

The first condition for an element to be a saddle price is that it should be the minimum element in the row and after that we check if it is the maximum element in the column or not.

Let‚Äôs start solving this problem and don't worry, this is not a difficult problem. Trust me, this is an easy one.

So, let us move step by step. As discussed, the first step is to find the minimum element in the row. Let‚Äôs work upon that.

Working :

We enter a row and assume that it‚Äôs first element is the minimum element. Then as we move further in the row, we compare our assumed minimum element with every element and as we encounter a new minimum, we update our minimum.

The extra colno is to remember what was the column number of our minimum element.

Let‚Äôs consider that we have found the minimum element of the First row. According to the definition of sadness price, now we need to determine if it is the maximum element of its column.

If yes, then it is the saddle price of this given matrix, else we will have to move to the next row and repeat the same process again. This working is described briefly in the solution video from . >

Time Complexity :

In the worst-case scenario we might need to traverse all the elements in the matrix, so the time complexity comes out to be n square (n*n).

Space Complexity :

Since we are using only a handful of variables, space complexity is constant, O( 1 ).

Now, there is a fun fact, that in a matrix there can never be more than 1 saddle price.

Let‚Äôs see how.

In the given figure, we have assumed that we have 2 saddle prices i.e. H and J. Since they are saddle prices, they are smallest in the row and largest in the column.


Thus, H is greater than F, which is a row element and smaller than L, which is another column element. Same for J, J > F and J < L. Putting the 2 inequalities of H together we get, L < H < F ‚áí L < F. And similarly putting together the 2 inequalities of J we get, F < J < L ‚áí F < L.
